# 🏠 House Price Prediction (ML Project)

## 📌 Overview
This project predicts house prices using regression techniques like Random Forest and XGBoost.

## ⚙️ Features
- Data preprocessing
- Handling missing values
- Skewness transformation
- Feature encoding
- Hyperparameter tuning
- Model evaluation (RMSE, MAE, R²)

## 📊 Models Used
- Random Forest Regressor
- XGBoost Regressor

## 📈 Evaluation Metrics
- RMSE
- MAE
- R² Score
